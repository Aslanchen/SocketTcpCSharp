# SocketTcpCSharp

语言：C#

工具：VS2015

版本：1.0.0

Socket Tcp C#版本，客户端以及服务端的简单封装

1：客户端发送和接收。

2：服务端发送和接收。

3：发送和接收队列处理。

4：数据的分发，MsgCenter.cs，根据具体情况做参考。

5：Tcp粘包处理，本程序默认以数据前4位作为长度判断，请根据实际情况修改 SocketClient 和 SocketServer 的 OnReceive  函数。

6：生成数据的时候，请注意数字类型的高低位。C++和JAVA是由低到高，C#是由高到低，请自行修改。
